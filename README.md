Project Workflow or Development Workflow.
CompareBuddy follows a milestone-based development process.

### Milestone 1 – Project Foundation
- Brand Guide
- Website Blueprint
- Design System

### Milestone 2 – Homepage Master Design
- High-fidelity visual design
- User experience approval
- Brand consistency review

### Milestone 3 – Website Development
- GitHub implementation
- Responsive layouts
- Component development

### Milestone 4 – Quality Assurance
- Testing
- Performance optimisation
- Accessibility
- SEO

### Milestone 5 – Launch
- Production deployment
- Domain configuration
- Analytics
- Monitoring

## Project Rule

**Nothing gets built until it has been approved in the Design Milestone.**

All implementation must follow the approved design documentation. Any design changes must be approved before development begins.
