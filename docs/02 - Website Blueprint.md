# Purpose

This document defines the complete structure of the CompareBuddy website before any design or development begins.

It establishes:

- Website pages
- Navigation
- User journeys
- Content hierarchy
- Calls-to-action
- User experience principles

The objective is to create a website that presents CompareBuddy as a premium desktop software solution for South African insurance professionals.
